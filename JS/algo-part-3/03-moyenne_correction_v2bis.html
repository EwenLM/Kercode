#03 - La moyenne

Proposez successivement deux écritures pour la function average qui calcule et affiche une moyenne des notes données dans le tableau du même nom.
const notes = [8.75, 14, 13, 5.5, 9, 17, 11.25]
La moyenne sera arrondie à 2 chiffres après la virgule.

Les 2 méthodes :

  • avec une boucle for ou for...of
  • avec la méthode forEach()

Si tout se passe normalement :

average (notes) affichera 11.21
